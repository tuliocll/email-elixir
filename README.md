## Como usar ❓

Clone o projeto:

```bash
git clone https://github.com/tuliocll/email-elixir.git
```

Configure o SMTP em `config/config.exs` (recomendo mailtrap para teste).

Instale as dependências:

```bash
mix deps.get
```

Rode o projeto

```bash
iex -S mix
```

Acesse a url: [http:localhost:8085](http:localhost:8085).
